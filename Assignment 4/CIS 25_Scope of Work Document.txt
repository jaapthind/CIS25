1. Project Overview
This project is a text-based productivity app built in C++ that turns real-life tasks into a game. Users earn experience points and level up a character profile when they complete their daily goals. The application saves all character stats and habits to local text files so the data isn't lost when the app closes. 
2. Features
* Text-Based Main Menu: A simple terminal screen that displays character stats (Level, XP, and Gold) and a list of active habits.
* Habit Management: Ability to add new habits, edit existing ones, and delete completed tasks.
* Leveling System: Logic that rewards the player with experience points and levels them up as they complete goals.
* Character Data Saving: Long-term storage using a text file (user_profile.txt) to save the user's level, XP, and gold when they quit.
* Habit List Saving: Long-term storage using a text file (habits.txt) that automatically saves active habits and loads them back in when the app starts.
